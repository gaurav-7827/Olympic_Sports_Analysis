# Olympic Sports Analytics Project

## Project Overview
This project analyzes historical Olympic Games data to uncover trends in participation,
performance, and medal distribution across regions, sports, and time.

The objective of this project is to derive meaningful insights using data analysis
and visualization techniques and present them in an interactive and easy-to-understand format.

## SQL Analysis
SQL queries were used to:
- Identify dominant regions and sports
- Analyze medal distribution patterns
- Track participation growth and decline
- Study regional impact on overall medal tally
- Perform joins across multiple Olympic datasets

---

## Conclusion
The analysis highlights that certain regions consistently dominate specific sports,
while overall Olympic participation has grown more inclusive over time.
The project demonstrates how data analytics tools like Excel, SQL, and Power BI
can be effectively combined to extract insights from large real-world datasets.
